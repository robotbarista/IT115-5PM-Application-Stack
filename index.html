MERN Stack & Tutorial

MERN is an acronym for MongoDB, Express, React, and Node, and they are the key technologies that make up this particular stack.

The MERN stack provides the opportunity to create 3-tier architecture consisting of a frontend, backend and database exclusively using JavaScript and JSON. It improves the end users experience and is utilized for easier and faster deployment of full stack web applications. The languages technologies that MERN utilizes being JavaScript means it is also an easier stack to learn.

To break down the particular components that make up this stack:

Mongo DB is a document database that is non relational and it contains documents. They are stored with like documents, and the system is scalable and flexible.

Express.js is a server side framework that makes it easy to connect to databases like Mongo DB and helps the Node and web application fast and seamless. These components are on the application tier.

Node.js gives you the ability to to execute JavaScript outside a browser. It is not a framework or a browsing language. It provides web server functionality.

React is a front end Javascript library/ framework. It gives you the tools to build your application.
